# Spec documents

The protocol is specified across a numbered series of Markdown documents in
[`spec/`](https://github.com/random-walks/rrvix/tree/main/spec). Each document
has a stable filename prefix (`0001-`, `0002-`, ...) so it can be referenced
from RRPs and code by ID.

!!! warning "M0.4 in progress"
    Spec documents are being written as part of milestone M0.4. Until they
    land, the [whitepaper](../protocol/whitepaper.md) is the authoritative
    description of the protocol design.
