# Security policy

**Please don't open a public GitHub issue for security reports.** This repo's
schemas and spec are consumed downstream (rrxiv-python, web clients, running
instances), so vulnerabilities should reach the maintainer privately, before disclosure:

- **Email**: <albisburdige@protonmail.com>
- **Subject line**: `[security] rrxiv — <short description>`

You'll get an acknowledgement within 72 hours and an initial triage within 7 days.
